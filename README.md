## How to run
1. Clone this rep.  
`git clone https://github.com/elisioff/end-poo.git`  
  
2. Change working directory to end-poo/end-poo  
`cd end-poo/end-poo`  
  
3. Compile  
`javac *.java`  
`javac Exceptions/*.java`  
  
4. Run  
`java Main`
